  •   Some Basics
  • The following bits of pieces and commands would be useful in the upcoming tutorials:

    In Python 2.7, print statements don't require a paranthesis, while Python later than 2.7 require parenthesis in print. Keep this in mind.
    This tutorial is based on Python 2.7.

         Hit Enter after every line of code to execute it.

         Python Files are saved using the .py extension.

         Type exit() in the IDLE to close Python.

         Comments are created using #, they don't execute on run! And are only visible to the people seeing the actual code. In the tutorials, the comments will be used to guide you.

         In the tutorials, lines that will have >> in their beginning will be the code and the ones without them will be the output.
