Conditionals



Sometimes in a code, one likes to do things based on conditions. One such profound example is, say in a program to check which of the two numbers are greater. We compare two numbers and print which one is bigger.

This is called a conditional operation, and it is done using if, else, and elif (Short for else-if) statements, which form the basics of coding methodoligy).

Take for example this code snippet :

>> if(5 < 7):
     print "This will always come"
>> else:
     print "This will never come"
As in the example above. The condition inside the if-else statements are enclosed in brackets and the part to be executed has to be indented using a ':' properly. This goes for every piece of code in Python.

Note : If in the program one of the conditions is met, then the other condition is not taken care of. Keep this in mind as it is really important and a topic of great confusion for beginner coders.

Another code snippet is as follows :
>> x = input("Enter the first number : ")

>> y = input("Enter the second number : ")

>> if(x == y):       # The == operator is used to compare quality between two 'operands'
     print "The two are equal."
>> else:
     print "The two are unequal."

The Elif Statement

Life is already restricting enough, the last thing you would actually want is coding not giving you enough options. Well, there is no lack of coding options.

Even in the conditionals in python, you can have as many conditions as you want, using the elif statement.

Take for example this code snippet and learn it the final way for this page :
>> x = input("Enter the first number : ")

>> y = input("Enter the second number : ")

>> if(x == y):
     print "The two are equal."
>> elif(x > y):
     print (x," is greater.")
>> elif(x < y):
     print (y," is greater,")
>> else:
     print "This will never show."
Take a moment as to why the last statement will never work!
