Lists



As a specie, we want to get the most out of every single thing. Right? Similary, in programming, the main goal is to get the most out of every single thing we do, in the least amount of code.

Here, Lists come into play.

Lists are basically variables that can hold more than one value. And that value can be anything, any integer, floating constant or a string.
>> a = [5,6,"Person",8]
In the above, there is a list named 'a' which contains the values as shown in the '[...]'.

Printing Elements of a list

Indexing of any element in a list, starts from zero. You could just type :
>> print a[0]
5
>> print a[1]
6
>> print a[2]
Person
>> print a[3]
8
This could be done using for and while loops too (As said in the previous tutorial) :
>> for element in a:
       print element

#-------OR-------

 >> i = 0

>> while (i < len(a)):    # len() gives the length of array.

       print a[i]

Basic Operations on Lists

Append : (Adding element to the end of array) : a.append(< element you want to append >)

Ex : a.append("Hey This will be the last element!")

Insert at a given position : a.insert(< index >,< element >)

Ex : a.insert(2,3)     #Inserts 3 at index number 2, i.e: 3rd element.

Pop/Remove element from a given position : a.pop([1]) will remove 6 from the list a.

Reverse a list : a.reverse()

This is enough for lists in Python.
