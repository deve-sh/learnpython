Input-Output



Now that variables are out of the way. It would be so boring if one was left with a value of a variable that the programmer has set. Instead, it would be more fun if we had the option of the user had the option to enter the value of a variable!

To counter this issue; python comes built in with commands to input and print the value of the variable being used.

For Ex :
>> x = 5       # This would be boring, let's do something nice!
Here, x has a value that cannot be changed by the user. In order for the user to give the input. The Syntax will be :
>> x = input ("Enter the value for x : ")       # Perfect!

>> x = raw_input("Enter the string value of x : ")       # If x is meant to be a string


Let's learn what each line of it did :

  • input - This is used when the variable for whom the input is being taken; is meant to be a numeric value (Integer or Float). Ex: 5 or 5.0

  • raw_input - This is used when the variable for whom the input is being taken; is meant to be text. Ex : "I am someone" is a string and not a number. Similarly "5" and 5 are different, the first is a string (text), whereas the latter is a number (Interger).

  • The Message - This will pop up on the screen when the user is about to enter the value of the variable.

Output

Printing a variable to the screen is even more easy! Use the print function! (Don't worry if you don't know what a function is).

Here, let's try it :
>> x = 5

>> print x      # Prints 5 to the console!

5

>> y = 5.1

>> print y      # Prints 5.1 to the console!

5.1

>> z = "Hey There!"

>> print z      # Prints 'Hey There!' to the console!

Hey There!

You could also type in things directly to print them to the console.
>> print 5 + 6      # Prints 11
11
You can also connect strings 1 by 1 or print multiple things at once by using bracks and commas :
>> print("The answer to everything is ", 42)
The answer to everything is 42      # This is the output
