Operators



Now that input and output is out of the way too. Let's look at what we can do with our variables. Or rather, how?!

Python comes built in with expressions that allow you to change (++ , -- , *) , modify(+ , - , * , / ) and assign ( = ) values concerning variables.

For Ex :
>> z = x + y       #The equals sign is an operator that is working on three vars.
Here, z has a value that is equal to the sum of x and y. The + is an operator, similarly the = is also an operator. More examples are :
>> z = 7/8 # / is the division operator.

>> y = 9*8 # * is the multiplication operator.
The above are called mathematical operators.

There are the following main type of operators :
  • Mathematical - Used to perform mathematical tasks.I.E: + , - , * , /


  • Assignment - These are used to assign particular values to a set of variables.I.E = (equals), != (Not eual to) etc.


  • Logical - It would be better to learn them by practical application, next topic covers them. In all they work to give meaning to a statement or check whether a statement is true or not, or just comparing the values of two variables. They include == (If equal), != (If not equal), > (Greater Than), < (Less Than)
  • Remainder (%) - This tells what the remainder is when a number is divided by another number. I.e. 5%2=1 and 4%2=0, if the remainder is 0, the number is divisible by the other number.

Operators are often used in accordance with each other, ex: >=(Greater than or Equal to), <=(Less than or Equal to); or a number of operators can be used in a single statement in any programming language.
