Variables



In Python, data is stored in the form of variables.
Variables are data with or without a value whose values can be changed during the run of a program.

For Ex :
>> x = 5
Here, x is a variable.

There are a few types of variables that can be used in a Python program. Most notably :

  • Integer - These variables store numerical values without a decimal point.


  • Float - These variables store numerical values with a decimal point.


  • Strings - These variables store text values, are declared inside "...".

In Python, unlike other programming languages (Skip this if you don't know any other programming language!), you don't have to define which type of data is being stored in a variable.

Defining / Declaring a variable

A sample definition of the different types of variables :
>> x = 5      #This is an integer type variable!

>> y = 5.1      #This is a floating type variable!

>> z = "Hey There!"      #This is a string!

Note : The values of a variable CAN be changed during the program. This is the most fundamental concept of programming.
>> x = 7

>> x = x + 1

>> print x

8

>> x-= 1     # Same as x=x-1

 >> print x

7
As you can see in this example, the value of x has been increased by one, and that is the further default value of x from that point, and then it was decreased by one. That is it for the variables in Python.
