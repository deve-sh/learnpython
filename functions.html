Functions



Think of a scenario again. This time a practical one. Suppose you have been alloted a work and you are being paid a thousand dollars for it; the catch is, the work takes 5 days to complete; whereas the client needs it done by tomorrow. So unless you are Bruce Wayne; What do you do? Out of sheer reluctance; you will have to hire a few people, pay them and get the work done otherwise the thousand dollard are lost too and you would have wasted those 24 hours you could have spent doing something else, much more useful!

In this example, in programming you are the the main code/whole program and those people you hire are functions. You give them something to do and they do it for you. This is the very definition of a function.

A function is a part of code that can take a command, work on it, and return it for you.

For ex: Let's say I want to square a number; but am too lazy to do it. So I just give the function a number, say 5, and the function would give me 25. This is the example of an argument/Parameter. Let's look at how it works:
def function(< argument list >):     # Or just any function name
    < All the code goes here >
Here def stands for define. If you didn't know, which you did. So congrats.

An example of a function is :
def function():
    print "Hey There!"

Calling a function

Just like you can ask the person you hired about what the result he had gotten. You can call a function to execute its code like this.
# < function name >()

>> def function():
    print "Hey There!"

>> function()     # This will call the function and the code will be executed.

Passing a parameter

Just like asking someone the square of a number, notice, you are giving them a number to work on. Right?

Similarly, you can give anything to a function to work on.

Just look at this :
>> def square(x):   # x can be any value you may pass in the function
    print x*x
>> square(6)
36
